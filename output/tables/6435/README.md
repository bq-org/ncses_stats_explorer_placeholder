**Type:** RH (Row Hierarchical)

## Notes

Because of rounding, detail may not add to total. Only those agencies and subdivisions that had outlays in variables represented by this table appear in the table. FYs 2020 and 2021 outlays include additional funding provided by supplemental COVID-19 related appropriations (e.g., Coronavirus Aid, Relief, and Economic Security [CARES] Act). See technical table A-2 for additional notes associated with the agencies listed in this table.
